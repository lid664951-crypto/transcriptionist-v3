# HarmonyOS Sans 字体

请从以下地址下载 HarmonyOS Sans SC (简体中文) 字体文件：

## 下载地址

1. 华为官方: https://developer.harmonyos.com/cn/design/resource
2. GitHub 镜像: https://github.com/AiursoftWeb/HarmonyOS-Sans/tree/main

## 需要的字体文件

请将以下字体文件放到此目录：

- `HarmonyOS_Sans_SC_Regular.ttf` - 常规字重 (正文)
- `HarmonyOS_Sans_SC_Medium.ttf` - 中等字重 (标题)
- `HarmonyOS_Sans_SC_Bold.ttf` - 粗体 (强调)
- `HarmonyOS_Sans_SC_Light.ttf` - 细体 (注释) [可选]

## 许可证

HarmonyOS Sans 字体采用 SIL Open Font License 1.1 许可证，可免费商用。
